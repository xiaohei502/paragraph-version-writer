# paragraph-version-writer
段落版本写作器

**段落版本写作器** 是一个轻量级、网页端的写作辅助工具，专为小说或创作文本设计，让你按段落管理内容，每段可以保存多个版本，方便修改、优化与迭代。

## 功能特点

- **段落管理**：将文本按段落组织，每段可创建多个版本。
- **版本控制**：每段可保存多个历史版本，并可以设置某个版本为“展示版”。
- **文本导入导出**：
  - 导入 TXT 或源文件
  - 导出文本 (`TXT`) 或 Markdown 格式 (`Markdown`)
  - 导出源文件 (`JSON`) 以便下次继续编辑
- **持续写作支持**：
  - 可在当前段落或文末新增段落，保持创作连续性
  - 自动保存草稿和展示版切换
- **自适应界面**：支持电脑端与手机端显示

## 使用方式

1. 下载或打开 `index.html` 或 GitHub Pages 链接。
2. 导入文本或源文件。
3. 对每个段落进行编辑或新增版本。
4. 设置某个版本为展示版（会自动保存）。
5. 完成后导出 TXT、Markdown 或 JSON 文件。
6. 导出 JSON 文件后，下次再导入JSON 文件会回溯上次工作台的所有状态，以便于后续编辑文本。

## 部署

- 可直接在本地打开 `index.html` 使用。
- 可部署到 GitHub Pages 或其他静态网页服务器，直接在线使用。

## 适用场景

- 小说创作和文本写作
- 文本版本迭代和优化
- 写作练习和创意实验
- 给文字段落添加上其它信息（比如备注、提示词）
